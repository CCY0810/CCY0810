### iPixelo – Image Processing & Enhancement Application
🔗 https://github.com/CCY0810/iPixelo

A mobile-inspired image editing and enhancement application that demonstrates the practical implementation of fundamental image processing techniques through an interactive GUI.

**Key Features**
- Geometric transformations including **cropping, rotation, and flipping**  
- Adjustable **brightness, blur, noise reduction, sharpness, and saturation**  
- **Region of Interest (ROI) processing** for localized image enhancement  
- Multiple **visual filter presets** such as grayscale, bright, and cool tone filters  
- **Text insertion tool** with customizable position, color, and rotation  
- Interactive parameter control using sliders for real-time adjustments  
- Non-destructive editing workflow with **image reset functionality**  
- Export and save edited images to local storage 

**Technologies:** Python, OpenCV, PyQt6
